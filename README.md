# heroku-de-ruboty
Ruboty on Heroku

# Tips
- the name of bot can be defined by ENV
    - `ROBOT_NAME`
    - default name is `ruboty`
- say method
    - do reply
        - `message.reply`
    - username
        - `message.from_name`
    - thw reply example
        - `message.reply("@#{message.from_name} 🍻🍻🍻🍻🍻)`
